# php gallery

simple image and video gallery on PHP

### How to install
create DB from dump in base.sql
configure public/core/config.php

for upload video need ffmpeg
```sh
$ sudo apt update
$ sudo apt install ffmpeg
```

need enable "shell_exec" php function
